2. Install dependencies:
   pip install -r requirements.txt
3. Run the game:
   python main.py


## Known Issues ❌
- Camera initialization may fail if no webcam is detected.
- Sound files may not load if paths are incorrect.

## Controls 🎮

### Keyboard:
- ⬅️ / ➡️ – Move piece left/right
- ⬇️ – Hard drop
- ⬆️ – Rotate piece
- Esc – Quit

### Camera-based (with object tracking):
- Move pink object left/right – Moves piece
- Rotate object – Rotates piece
